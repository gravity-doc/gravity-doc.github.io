我们引入了官方的 **Jupyter AI** (`jupyter-ai`)。您可以在左侧边栏找到 "Jupyternaut" (🚀) 图标。

## 🍎 6. 科研绘图福利：中文字体支持

系统已预设常用的科研字体，解决 Matplotlib 绘图时中文显示为方块的问题。

* **默认黑体**: `SimHei` (支持 `plt.title('中文')`)
* **论文标配**: `Times New Roman` + `SimSun` (宋体)
* **设置方法**: 默认已生效。如需指定宋体，在代码开头加入 `plt.rcParams['font.serif'] = ['SimSun']` 即可。

## ⚙️ 个性化设置 (Settings)

点击顶部菜单 **Settings** -> **Settings Editor** 即可调整插件行为。

> [!WARNING]
> **如何彻底关闭语法检查？**
> 在 `Language Server` 的 User Preferences 中填入：
> ```json
> { "client": { "diagnostics": { "enabled": false } } }
> 
> ```
