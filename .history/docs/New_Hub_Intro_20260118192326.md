#### 2. ‚ö° One-Click DeepSeek Integration (Auto-load)

To avoid re-entering your API credentials every time you restart your environment, you can create an **IPython Startup Script**. This script automatically sets your environment variables and loads the necessary AI magic commands upon startup.

Run the following block in your **Terminal** (Note: Replace `YOUR_API_KEY` with your actual key):

```bash
# 1. Ensure the IPython startup directory exists
mkdir -p ~/.ipython/profile_default/startup/

# 2. Create the initialization script to automate DeepSeek setup
cat <<EOF > ~/.ipython/profile_default/startup/01-jupyter-ai-init.py
import os

# Configure environment variables for the DeepSeek API
os.environ["OPENAI_API_KEY"] = "YOUR_API_KEY"
os.environ["OPENAI_API_BASE"] = "https://api.deepseek.com/v1"

try:
    # Automatically load Jupyter AI magic commands
    ip = get_ipython()
    if ip:
        ip.run_line_magic('load_ext', 'jupyter_ai_magics')
        # Hardcode the default model to DeepSeek
        ip.run_line_magic('config', 'AiMagics.default_language_model = "openai-chat:deepseek-chat"')
    print("‚úÖ Jupyter AI initialized: DeepSeek is ready!")
except Exception as e:
    print(f"‚ùå Initialization failed: {e}")
EOF

```
